# AppLocker Pro

Aplicación de seguridad desarrollada en Python que permite proteger aplicaciones mediante contraseña, evitando accesos no autorizados.

## Características

• Bloqueo de aplicaciones mediante contraseña  
• Interfaz moderna y fácil de usar  
• Protección en segundo plano  
• Compatible con Windows  
• Ligero y rápido

## Instalación

Opción 1 – Ejecutar versión compilada:

Descargar el archivo .exe desde la sección Releases.

Opción 2 – Ejecutar desde Python:

Requisitos:

Python 3.10 o superior

Instalar dependencias:

pip install -r requirements.txt

Ejecutar:

python main.py
